## 简介

项目使用 webpack 构建. 集成大量博客插件. 可以用它做以下三件事:

1. 安装: 在您的博客园安装这个项目中已经集成的皮肤.安装之后,可以快速切换其他皮肤.
2. 创建: 快速创建一个的博客园皮肤,通过打包生成文件,供您使用.
3. 分享: 快速创建一个博客园皮肤并将它贡献给项目,园友就能够切换到您的皮肤了.
